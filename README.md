# BlinkIT Grocery Sales Analytics — Power BI Dashboard

## Project Overview
Interactive Power BI dashboard analyzing BlinkIT grocery 
sales data covering product categories, outlet performance, 
sales distribution and customer ratings across multiple 
outlet types and locations.

## Dashboard Features
- Sales performance by product category
- Outlet type and size analysis
- Fat content distribution analysis
- Location-wise sales breakdown
- Customer rating insights
- Interactive slicers for dynamic filtering

## Dataset Details
- Product categories and fat content
- Outlet type, size and location
- Sales figures and ratings
- Item visibility metrics

## Tools Used
- Power BI Desktop
- Power Query (Data Cleaning)
- DAX (Data Analysis Expressions)

## Key DAX Measures
- Total Sales
- Average Sales
- Average Rating
- Total Items Count

## Data Cleaning Steps
- Removed duplicates
- Standardized fat content categories
- Handled null values
- Formatted data types

## Key Insights
- Top performing product categories
- Best performing outlet types
- Sales distribution by location tier
- Relationship between outlet size and sales
